脚本 (Scripts)

以下是项目中定义的 npm 脚本：

  • start: 启动项目主程序。 
    npm start
  • deepseekTest: 测试 DeepSeek API 的功能。 
    npm run deepseekTest
  • test: 运行单元测试并生成覆盖率报告。 
    npm test
  • test:watch: 实时监听文件变更并运行测试。 
    npm run test:watch
